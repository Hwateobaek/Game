# 반사신경 그리드 🎯

반짝이는 칸을 최대한 빠르게 누르는 30초 반응속도 게임. 의존성 없는 단일 HTML 파일입니다.

## 플레이

- **로컬:** `index.html` 더블클릭
- **온라인:** https://hwateobaek.github.io/tesmparary/

## 기능

- 30초 한 판, 칸이 무작위로 반짝이면 클릭
- **점수 = 반응속도 × 콤보 배율 × 난이도 배율** (빠르고·연속으로·어렵게 맞힐수록 고득점)
- **콤보 보너스** — 연속 명중 시 배율 상승(최대 2.5×), 빈 칸 클릭 시 콤보 끊김 + 감점
- **3×3 / 4×4 / 5×5 🔥** 난이도 (난이도 높을수록 점수 배율↑)
- **랭킹** — 상위 기록을 `localStorage`에 저장·표시 (메달 표기)
- **효과음** — 명중(콤보 오를수록 피치↑) / 실수 사운드
- **배경음악** + 🔊 음소거 토글, 종료 시 엔딩 징글 (모두 Web Audio 실시간 합성, 저작권 없음)
- 모바일 반응형 (화면 폭에 맞춰 그리드 자동 조절)

## 배포 (GitHub Pages)

`https://hwateobaek.github.io/tesmparary/` 에서 플레이할 수 있습니다.
